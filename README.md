# 创意来源
在我们外出游玩时，我们经常会因为玩得过于激动而导致我们长时间滴水未进。水是生命之源，如果长时间缺水，身体就会感到不适。于是，我们崇文创新小队决定做一个智能水杯来提醒我们喝水。
# 创作设想
1.设计水杯整体尺寸进行初步制作；                                             
2.使用蜂鸣器来提醒喝水；                                          
3.用灯带显示喝水次数。                                      
4.使用Arduino主板和其他硬件组成硬件部分；                                        
5.使用mind+编程达到硬件功能；                                                
6.使用Laser Maker 进行外形加工；                                               
7.用雪弗板制作水杯主体并切割；
